# Coding Standards

## Frontend

- Follow the [Airbnb JavaScript Style Guide](https://github.com/airbnb/javascript)

---

## Backend (Java)

- Use constructor injection — not field injection
- Keep services thin and focused — business logic only, no controller or persistence concerns

---

## Git
